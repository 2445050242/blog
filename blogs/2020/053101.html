    ==只能生成正式版==

    生成的三种方式：

    1、wxacode.createQRCode 适用于需要的码数量较少的业务场景。通过该接口生成的小程序码，永久有效，有数量限制 详情

    2、wxacode.get 适用于需要的码数量较少的业务场景。通过该接口生成的小程序码，永久有效，有数量限制 详情

    3、wxacode.getUnlimited 适用于需要的码数量极多的业务场景。通过该接口生成的小程序码，永久有效，数量暂无限制。链接后面不能带参数，参数只能放在scene属性中，且限制长度为最大32个可见字符 详情

    区别 createQRCode/get getUnlimited
    数量 有限，总共生成的码数量限制为 100,000 无限
    参数 放在path参数后，最大长度 128 字节 放在scene参数内，最大32个可见字符，只支持数字，大小写英文以及部分特殊字符
    适合场景 需要的码数量较少 需要的码数量极多

    3、绘制canvas 详情

    canvasToTempFilePath 把当前画布指定区域的内容导出生成指定大小的图片。在 draw() 回调里调用该方法才能保证图片导出成功

    /** 绘制canvas */
getCanvas = async (text, context) => {
    const that = this
    const { imgList } = this.state
    for (let i = 0; i < imgList.length; i++) {
        // 遍历渲染所需图片资源
        await this.drawImg(imgList[i], context)
        if (i === imgList.length - 1) {
            // 绘制文案
            context.fillStyle = '#ffffff'
            context.font = '12px 微软雅黑'
            context.fillText(text, 63, 26)
            context.draw(true, () => {
                Taro.canvasToTempFilePath({
                    height: 1334,  // 绘制的canvas高度
                    canvasId: 'canvas', // 画布标识，传入 canvas 组件实例
                    success(res) {
                        // canvas绘制成功执行回调，res.tempFilePath为canvas图片临时路径
                    },
                    fail(err) {
                        // 绘制失败提示
                        Taro.showToast({
                            title: '生成图片失败！',
                            icon: 'none',
                            duration: 2000
                        })
                    },
                    complete() {
                        // 绘制完成
                    }
                }, this.$scope) // 在自定义组件下，当前组件实例的this，以操作组件内 canvas 组件
            })
        }
    }
}

    • # 解决 react/vue 安卓分享链接会被忽略掉hash参数，导致分享页面全部跳回首页问题

    判断为安卓时分享链接添加share=true参数，然后在index.html中判断share参数为真时则自动拼接上真正需要跳转的链接地址

    // 获取链接参数
function getQueryString() {
    var reg = new RegExp(`(^|&)${'share'}=([^&]*)(&|$)`);
    var r = window.location.search.substr(1).match(reg);
    if (r != null) return unescape(r[2]);
    return '';
}

// 替换url
function urlReplace() {
    var url = window.location.href
    if (url.indexOf('share') > 0) {
        history.pushState('', 'Title', url.replace(/share=true/g, ''))
    }
}

// 判断为安卓分享时，替换url并跳转拼接后的真实分享地址
if (getQueryString('share') === 'true') {
    urlReplace();
    window.location.href = window.location.origin + '/udc/wap/sign-in/#/share';
}
