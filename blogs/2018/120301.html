Laya引擎踩坑

vuePress-theme-reco Escape    2018 - 2020

Laya引擎踩坑

Escape 2018-12-03 laya 游戏

  • # 细节问题

    • 图片宽度不能超过 2048

    • 路径不能使用 ./ ,并且区分大小写，不然在打包成微信小游戏会报错

    • 创建场景时需要选择为分离模式才能生成UI配置文件

    • 子域 bitmap._addReference() 报错，也是因为版本太老，需要更新到最新引擎。 或则自己手动加个判断，判下 bitmap._addReference 是否为 null

    • 绘制开放域内容时需要添加背景，否则会模糊

    • 使用了开放域组件后可能会产生掉帧问题（在安卓上表现比较明显，通常会掉到 20-25 帧）,所以尽量避免在游戏过程中使用开放域，需要在不使用开放域的时候停止开放域渲染具体做法分 1.0 和 2.0:

      1.0做法:

      在使用开放域过程中开启：

      //_subContextSprite是Laya.Texture类型元素，小程序使用，非常费
this._subTexture.bitmap.alwaysChange = true;
      1
      2

      在不使用开放域时设定：

      //_subContextSprite是Laya.Texture类型元素
this._subContextSprite.visible = false;
//_subTexture为子域画布元素,这里一定要设为false,不然会很卡
this._subTexture.bitmap.alwaysChange = false;
      1
      2
      3
      4

      2.0做法:

      在使用开放域过程中开启：

      //_my_open为2.0增加的开放域组件
this._my_open.visible = true;
      1
      2

      在不使用开放域过程中移除开放与组件：

      this._my_open.removeSelf();
      1

  • # 微信开放域创建和注意点：

    • 创建开放域

      1、创建一个新工程，作为开放域工程
      2、按照普通流程创建需要显示精灵等元素，并设置 visible = false
      3、接收主域发送的信息标志添加数据并显示对应的开放域元素

      <!--接收主域信息-->
Laya.Browser.window.wx.onMessage(data => {
	if (data.message == undefined) {
		return;
	}
	let jsonData = JSON.parse(data.message);
	<!--根据信息所带的不同标志来控制对应的开放域元素的行为-->
	switch (jsonData.tag) {
		case "hideRankView": {
			this.rank.close();
			break;
		}
		case "showRankView": {
			this.openid = jsonData.data.openid;
			this.getRankList()
			break;
		}
	}
});

    • 创建主域：

      1、创建离屏画布（1.0需要自己手动绘制离屏画布，2.0提供了微信开放域组件）:

      1.0绘制离屏画布

      <!--判断是否在微信小游戏环境中-->
if (Laya.Browser.onMiniGame) {
    <!--判断是否存在离屏画布-->
	if (this._subCount == 0) {
		Laya.timer.once(200, this, function (): void {
		    <!--创建离屏画布具体步骤-->
			this._subTexture = new Laya.Texture(Laya.Browser.window.sharedCanvas);
			this._subTexture.bitmap.alwaysChange = true;
			this._subTexture.mouseEnabled = false;
			this._subContextSprite = new Laya.Sprite();
			this._subContextSprite.mouseEnabled = false;
			this._subContextSprite.size( Common.Config.WIDTH, Common.Config.HEIGHT);
			this._subContextSprite.zOrder = 1000;
			Laya.stage.addChild(this._subContextSprite);
			this._subContextSprite.graphics.drawTexture(this._subTexture, 0, 0, Common.Config.WIDTH, Common.Config.HEIGHT);

			callback();
		});
	} else {
	    <!--若存在离屏画布则直接控制其显示-->
		this._subContextSprite.visible = true;
		this._subTexture.bitmap.alwaysChange = true;
		callback();
	}
}

    • 微信小程序配置：

      1、将开放域工程打包成微信小程序，记得勾选***是否为微信开放数据域项目***

      2、在主域的项目目录下新建开放域文件夹 openDataContext,文件夹名称随意，并将打包完的开放域项目代码复制到这个文件夹下，也可以在开放域项目打包时设定***发布目录***为这个文件夹，可以省去复制操作

      3、打开主域的game.json文件，并添加

      //openDataContext对应的是开放域所在的文件夹名称
"openDataContext": "openDataContext"
      1
      2

    • 注意点：

      • 开放域组件或离屏画布似乎不能绘制在弹窗上，所以只能直接添加到舞台当中并设定层级高一点

      • 在绘制开放域的过程中，若不涉及数据实时更新的问题，则可以提前绘制，只控制显隐，若涉及数据实时更新的问题，在主域控制开放域显隐之前先拿到最新数据并提前绘制好开放域元素，总的就是需要提前绘制。因为开放域获取数据和绘制需要一定的时候，如果在还没绘制完开放域，主域就控制开放域隐藏元素，会失效导致开放域不能被隐藏，出现bug
